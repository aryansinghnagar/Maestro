# Security Policy

## Supported Versions

We actively support security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| v1.1.x  | :white_check_mark: |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do **not** open a public issue. Instead, report it privately by emailing the maintainers at `asn.dyrnwyn@gmail.com`.

We will acknowledge receipt within 48 hours and provide a timeline for coordination and release of a patch.
