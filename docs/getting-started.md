# Getting Started

Follow this guide to get Maestro installed and configured on your machine.

## Prerequisites

- Python 3.11+
- A standard USB webcam or built-in camera

## Installation

Run the installation command for your platform:

### Linux
```bash
bash packaging/linux/install.sh
```

### macOS
```bash
brew install python
pip install gesture-controller
```

### Windows
Run the `windows_installer.nsi` generated package.
